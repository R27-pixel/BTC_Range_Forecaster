Model Architecture
No-Peek Guarantee
Walk-forward backtest: prediction at bar N uses strictly bars 0…N−1. Zero data leakage.
Volatility Clustering
σ estimated from a rolling 24-bar window. Calm periods → narrow range. Volatile periods → wide range.
Fat Tails
GBM driven by t(ν) innovations, ν fitted empirically. Captures BTC's heavy tail behaviour without over-widening ranges.
Monte Carlo
10,000 simulated 1-step paths. 95% CI from [2.5th, 97.5th] percentile of terminal prices.
Winkler Score
Penalises both width and misses. Score = width if correct; width + (2/α)·miss_distance if wrong. Lower is better.
